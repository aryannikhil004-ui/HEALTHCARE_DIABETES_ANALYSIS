## 🚀 Project Overview

This project presents a complete **data analytics workflow** on a healthcare diabetes dataset. It covers data preprocessing, exploratory data analysis (EDA), SQL-based insights, and an interactive Power BI dashboard to uncover patterns related to diabetes.

---

## 🎯 Objectives

* Understand key factors influencing diabetes
* Perform structured data cleaning and analysis
* Build meaningful visualizations for decision-making
* Create a professional dashboard for stakeholders

---

## 🧹 Data Cleaning (Python)

✔ Loaded dataset using Pandas
✔ Handled missing values & duplicates
✔ Outlier detection using IQR method
✔ Data type correction & formatting
✔ Prepared analysis-ready dataset

---

## 📊 Exploratory Data Analysis (EDA)

* Distribution of diabetic vs non-diabetic patients
* Age group analysis and trends
* Correlation between:

  * Glucose level
  * BMI
  * Insulin
* Visualizations using Matplotlib & Seaborn

---

## 🗄️ SQL Analysis (SQL Server)

* Imported cleaned dataset into SQL Server
* Key queries performed:

  * Total patients by outcome
  * Average glucose by age group
  * BMI vs diabetes relationship
* Used:

  * `GROUP BY`, `JOIN`, `WHERE`, `AVG`, `COUNT`

---

## 📈 Power BI Dashboard

✨ Features:

* Diabetes distribution chart
* Age Group vs Diabetes Patients
* KPI cards for key metrics
* Interactive slicers & filters
* Clean and professional layout
